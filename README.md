# vedeshpadal.me

My personal website built with [Astro](https://astro.build) and the [AstroPaper](https://github.com/satnaing/astro-paper) theme, inspired by [Steipete](https://steipete.com/)'s site.

## Features

- **Minimal & Fast**: Built with Astro for zero-JS by default.
- **Steipete-Style Thumbnails**: Clean, responsive blog post cards.
- **Image Zoom**: Lightbox-style zooming for better readability on all devices.
- **Dark Mode**: First-class support for light and dark themes.
- **Type-safe Content**: Using Astro Content Collections.

## Tech Stack

- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Content**: Markdown / MDX
- **Deployment**: GitHub Pages / EC2
